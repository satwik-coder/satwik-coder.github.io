## ⚙️ Usage Instructions
After installation, you can start using nullsec-netprobe to scan your network. 

### 👥 Scanning Your Network
1. Open a terminal or command prompt.
2. Navigate to the folder where you installed nullsec-netprobe.
3. Use the following command to start scanning:
